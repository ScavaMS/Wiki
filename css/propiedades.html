Propiedades

Propiedades de texto

color: Define el color del texto.
font-family: Especifica la fuente utilizada para el texto.
font-size: Define el tamaño del texto.
font-weight: Establece el grosor o peso de la fuente.
text-align: Alinea el texto horizontalmente en el elemento.
text-decoration: Agrega decoración al texto, como subrayado, tachado, etc.
text-transform: Transforma el texto en mayúsculas o minúsculas.
line-height: Establece la altura de línea del texto.
letter-spacing: Controla el espaciado entre caracteres.

Propiedades de fondo y borde

background-color: Define el color de fondo del elemento.
background-image: Especifica una imagen de fondo para el elemento.
background-size: Controla el tamaño de la imagen de fondo.
background-position: Define la posición inicial de la imagen de fondo.
border: Establece los estilos, anchos y colores del borde.
border-radius: Controla el radio de las esquinas del borde.
box-shadow: Agrega sombra alrededor del elemento.

Propiedades de diseño y posicionamiento

width: Establece el ancho del elemento.
height: Define la altura del elemento.
margin: Controla los márgenes externos del elemento.
padding: Define los espacios internos del elemento.
display: Especifica el tipo de visualización del elemento (bloque, en línea, etc.).
position: Controla la posición del elemento en relación con otros elementos.
top, right, bottom, left: Controlan el posicionamiento exacto del elemento.
float: Hace que un elemento flote hacia la izquierda o la derecha.

Propiedades de animación y transición

transition: Establece la transición suave entre los cambios de propiedad.
animation: Define una animación personalizada.
transform: Aplica transformaciones como rotación, escala, sesgado, etc.
opacity: Controla la opacidad del elemento.

Propiedades de diseño responsivo

media queries: Permite aplicar estilos diferentes según el tamaño de la pantalla o el dispositivo.
flexbox: Proporciona un sistema flexible de diseño en una dimensión.
grid: Permite crear diseños de múltiples columnas y filas.
