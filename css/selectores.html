Selectores

Selector de etiquetas

Selector de etiquetas: Selecciona todos los elementos que coinciden con la etiqueta especificada, como párrafos (<p>), encabezados (<h1>, <h2>, etc.) o elementos de lista (<ul>, <ol>, <li>).
Selector de clases: Selecciona todos los elementos que tienen asignada una clase específica. La clase se define en el atributo class de un elemento HTML y se utiliza para aplicar estilos a uno o varios elementos. Como .mi-clase.
Selector de ID: Selecciona un elemento único que tiene asignado un ID específico. El ID se define en el atributo id de un elemento HTML y debe ser único en toda la página. Como #Mi-ID.
Selector de atributo: Ejemplo: [type="text"]. Selecciona elementos que tienen un atributo específico y un valor correspondiente. Puedes especificar cualquier atributo y su valor entre corchetes, como [href] o [data-valor].
Selector descendiente: Ejemplo: div p. Selecciona todos los elementos p que son descendientes (hijos, nietos, etc.) de un elemento div. Este selector busca en toda la jerarquía del documento HTML.
Selector de hijo directo: Ejemplo: div > p. Selecciona todos los elementos p que son hijos directos de un elemento div, es decir, están inmediatamente anidados dentro del elemento div.
Selector de pseudo-clase: Ejemplo: a:hover. Selecciona un elemento cuando se encuentra en un estado específico. Las pseudo-clases comunes incluyen :hover (cuando se pasa el cursor sobre el elemento), :active (cuando el elemento está siendo activado) y :focus (cuando el elemento tiene el foco).
Selector de pseudo-elemento: Ejemplo: ::before. Selecciona una parte específica de un elemento y le aplica estilos. Los pseudo-elementos más utilizados son ::before (inserta contenido antes del contenido del elemento) y ::after (inserta contenido después del contenido del elemento).
